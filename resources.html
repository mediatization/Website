Screensight

The app we use to take pictures of participant's phones. Currently we only support Android but are looking to expand to Apple. We are hoping to release on the Google Play Store befor ethe end of 2026, in the meantime you can acces the source code of our app from the repository here.

Cloud Functions

The Cloud Functions we have written for Microsoft Azure to handle uploading images from participants phones as well as downloading them to our DMPO. The repository can be found here.

Data Management Participant Onboarding (DMPO)

The tool used by our researchers to download/decrypt images and to compile the screensight app. We are currently working on adding OCR functionality to downloaded images as well as annotation tools to enable researchers to more effeciently look through and code images. We are also working towards making a release version to simplify setting up and using this tool. The source code can be found here.
